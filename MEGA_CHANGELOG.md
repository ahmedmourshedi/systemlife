# سجل تغييرات نسخة Mega

## 2.0.0

### إضافات الواجهة

- 28 صفحة مستخدم متقدمة جديدة.
- 7 صفحات أدمن جديدة.
- صفحة بحث موحد.
- صفحة استيراد وتصدير مع روابط API.
- اختصارات PWA.
- Sidebar قابل للتمرير بسبب كثرة الوحدات.

### إضافات قاعدة البيانات

- توسعة `database/mega-extension.sql`.
- دمج التوسعة داخل `database/schema.sql`.
- جداول جديدة للوقت، المشاريع، اليوميات، العلاقات، الوصفات، الديون، الأصول، السفر، القرارات، الأتمتة والتنبيهات.
- جداول أدمن لجودة البيانات، سجل التدقيق، النسخ الاحتياطي، القوالب، والأمان.
- سياسات RLS للجداول الجديدة.
- فهارس مهمة للأداء.

### إضافات تقنية

- `components/mega/module-page.tsx` لإنشاء صفحات CRUD سريعة من تعريفات الوحدات.
- `lib/mega/module-registry.ts` لتعريف الوحدات والحقول من مكان واحد.
- `lib/mega/data.ts` للقراءة والبحث والتصدير.
- `lib/mega/analytics.ts` لمعادلات تحليلية قابلة للتوسع.
- `lib/mega/import-mapping.ts` لتجهيز مطابقة CSV.
- `lib/mega/prompt-templates.ts` لقوالب مراجعة ذكية.
- `lib/mega/security-checks.ts` لقائمة أمان.

### بيانات وتوثيق

- بيانات أداء بحجم يقارب 101MB.
- توثيق المنتج.
- دليل المستخدم.
- دليل الأدمن.
- دليل قاعدة البيانات.
- دليل الأمان.
- خطة اختبار.
- خارطة طريق.
