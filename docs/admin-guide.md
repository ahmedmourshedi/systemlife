# دليل الأدمن

## مهام الأدمن الأساسية

1. ترقية مستخدم إلى أدمن عند الحاجة.
2. مراجعة سجل التدقيق.
3. فحص جودة البيانات.
4. إدارة قوالب النظام.
5. توثيق عمليات النسخ الاحتياطي.
6. مراجعة أحداث الأمان.
7. نشر إعلانات النظام.

## أفضل ممارسات

- لا تجعل كل المستخدمين أدمن.
- راجع `profiles` دوريًا.
- استخدم سجل التدقيق لأي تغيير حساس.
- لا تعدل سياسات RLS إلا بعد اختبارها.
- احتفظ بنسخة من `schema.sql` قبل أي تعديل كبير.
