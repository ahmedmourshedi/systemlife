# قوالب Prompts للمراجعة الذكية

هذه القوالب لا تستخدم API داخل المشروع. هي نصوص جاهزة يمكنك نسخها إلى ChatGPT بعد تصدير البيانات.

## مراجعة مالية

حلل بيانات المصروفات والدخل التالية، واستخرج:

1. أعلى بنود الصرف.
2. بند واحد يجب تخفيضه هذا الشهر.
3. نسبة الادخار.
4. 3 قرارات عملية للأسبوع القادم.

## مراجعة صحية

حلل بيانات الوجبات والوزن والمياه والنوم، واستخرج:

1. نمط الالتزام.
2. أكثر سبب يضعف الالتزام.
3. تعديل واحد بسيط للوجبات.
4. خطة أسبوعية مختصرة.

## مراجعة تعلم

حلل الكورسات والكتب والمهارات، واستخرج:

1. هل التقدم حقيقي أم مجرد مشاهدة؟
2. ما أفضل تطبيق عملي هذا الأسبوع؟
3. ما الذي يجب إيقافه؟
4. ما خطة 7 أيام؟
