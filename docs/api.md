# واجهات API الداخلية

## فحص الصحة

```text
GET /api/health
```

يرجع حالة التطبيق ومتغيرات البيئة الأساسية بدون كشف القيم السرية.

## التصدير

```text
GET /api/export
```

يرجع ملف JSON يحتوي على بيانات الوحدات المتقدمة المتاحة للمستخدم الحالي عبر RLS.

## ملاحظات

- لا توجد Service Role Key داخل هذه المسارات.
- المسارات تعتمد على جلسة المستخدم في Supabase.
- أي بيانات لا يسمح بها RLS لن تظهر في التصدير.
