# الاستيراد والتصدير

## التصدير

يوجد مسار API:

```text
/api/export
```

يعطي ملف JSON لبيانات الوحدات المتقدمة المتاحة للمستخدم الحالي.

## الاستيراد

الاستيراد الفعلي من CSV لم يتم ربطه بواجهة رفع بعد، لكن تم تجهيز:

- جدول `import_jobs`.
- صفحة `الاستيراد والتصدير`.
- ملفات بيانات أداء داخل `sample-data/performance`.

## طريقة مقترحة للاستيراد لاحقًا

1. رفع CSV إلى Storage.
2. إنشاء Import Job.
3. قراءة الأعمدة ومطابقتها مع جدول الهدف.
4. تنفيذ الإدخال على دفعات.
5. تسجيل الأخطاء داخل جدول جودة البيانات.
