# دليل PWA على الجوال

## iPhone

1. افتح الموقع من Safari.
2. اضغط مشاركة.
3. اختر Add to Home Screen.
4. افتح التطبيق من الشاشة الرئيسية.

## Android

1. افتح الموقع من Chrome.
2. اضغط القائمة.
3. اختر Add to Home Screen أو Install App.
4. افتح التطبيق مثل أي تطبيق.

## ملاحظات

- التطبيق يعمل كتجربة PWA وليس تطبيق Native.
- وضع Offline يعرض صفحة احتياطية.
- البيانات الحقيقية تحتاج اتصالًا بقاعدة Supabase.
- Push Notifications الحقيقية تحتاج تطوير إضافي.
