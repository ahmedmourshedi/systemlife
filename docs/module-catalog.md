# كتالوج الوحدات

## وحدات الإدخال السريع

- الالتقاط السريع: لأي شيء لا تريد نسيانه.
- صندوق المراجعة: لمعالجة العناصر الخام.
- البحث الموحد: للعثور على السجلات عبر الوحدات المتقدمة.
