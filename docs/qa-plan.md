# خطة الاختبار

## اختبارات أساسية

- تسجيل حساب جديد.
- تسجيل دخول.
- إنشاء أول سجل مصروف.
- إنشاء أول سجل وجبة.
- إنشاء أول مهمة.
- إنشاء هدف وربط مهمة به.
- إنشاء عنصر في الالتقاط السريع.
- البحث عن العنصر.
- تصدير JSON.
- فتح التطبيق من الجوال كـ PWA.

## اختبارات أدمن

- تحويل مستخدم إلى أدمن.
- فتح لوحة الأدمن.
- إضافة سجل تدقيق.
- إضافة فحص جودة بيانات.
- إضافة قالب.
- إضافة إعلان نظام.

## اختبارات RLS

- مستخدم A لا يرى بيانات مستخدم B.
- الأدمن يرى بيانات عامة حسب السياسة.
- المستخدم العادي لا يستطيع فتح `/admin`.
- المستخدم العادي لا يستطيع إدخال بيانات في جداول الأدمن.

## اختبارات أداء

- استيراد جزء من ملفات `sample-data/performance`.
- اختبار سرعة صفحة البحث.
- اختبار صفحة التقارير بعد إدخال بيانات كثيرة.
