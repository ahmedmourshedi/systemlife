# دليل المستخدم

## أول 7 أيام

### اليوم الأول

- سجل مصروفاتك فقط.
- أضف حسابًا ماليًا واحدًا.
- أضف أول هدف بسيط.

### اليوم الثاني

- أضف الوجبات والمياه.
- أضف الوزن.

### اليوم الثالث

- أضف الكورسات والكتب.
- سجل درسًا واحدًا وسجل قراءة واحدة.

### اليوم الرابع

- استخدم الالتقاط السريع.
- أضف 5 أفكار أو مهام خام.

### اليوم الخامس

- أضف مخطط اليوم وإدارة الوقت.

### اليوم السادس

- أضف الروتين والعادات.

### اليوم السابع

- افتح المراجعات الأسبوعية واكتب مراجعة قصيرة.

## قاعدة مهمة

لا تحاول ملء كل شيء في يوم واحد. النظام قوي لأنه يتراكم مع الوقت.
