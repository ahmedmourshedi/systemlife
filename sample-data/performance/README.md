# بيانات أداء كبيرة

هذه الملفات موجودة حتى يكون المشروع ضخمًا وواقعيًا لاختبار الاستيراد والتصدير والبحث والتقارير.

لا ترفع كل الملفات إلى Supabase في البداية. استخدم عينة صغيرة أولًا.

الملفات:

- `expenses_performance.csv`: مصروفات كثيرة.
- `meals_performance.csv`: وجبات وسعرات.
- `tasks_performance.csv`: مهام وحالات.
- `life_events_performance.jsonl`: أحداث حياة متنوعة بصيغة JSON Lines.

الغرض منها اختبار الأداء، وليست مطلوبة لتشغيل التطبيق.
